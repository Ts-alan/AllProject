Модуль:
NetTest_02

Проект:
Центр Управления Vba32 с веб-интерфейсом

Выполняемые действия:
Отсылка на сервер пакета большого размера (до 32767 байт). Формат пакета формируется в соответствии с требованиями к серверной части ЦУ (первые два байта - размер пакета). Содержимое пакета формируется случайным образом.

Использование:
NetTest_02.exe IP_адрес_сервера порт_сервера
