Модуль:
NetTest_04

Проект:
Центр Управления Vba32 с веб-интерфейсом

Выполняемые действия:
Отсылка клиенту (Агенту) указанного файла, размером не более 65400 байт, подписанного ЭЦП (библиотека libcrypt). Закрытый ключ для подписи берется из реестра, раздел HKLM\Software\Vba32\ControlCenter\Signature\ (если установлен ЦУ, утилита возьмет его ключ).

Ахтунг! Не работает под x64

Использование:
NetTest_04.exe IP_адрес_агента порт_сервера имя_файла_для_отсылки
