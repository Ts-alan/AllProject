Модуль:
NetTest_03

Проект:
Центр Управления Vba32 с веб-интерфейсом

Выполняемые действия:
Отсылка на сервер указанного файла, размером не более 65533 байт. Формат пакета формируется в соответствии с требованиями к серверной части ЦУ (первые два байта - размер пакета). Для корректного парсинга бизнес-логикой ЦУ содержимое файла должно быть в кодировке UTF-8, но без первых трех байт (файл можно сохранить в Блокноте с кодировкой UTF-8, затем открыть по F4 в FAR'е и удалить первых три байта).

Использование:
NetTest_03.exe IP_адрес_сервера порт_сервера имя_файла_для_отсылки
